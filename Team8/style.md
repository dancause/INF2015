#Normes de programmation de l'équipe 8
##Langue
Français

##Nom des variables
snake_case

exemple: valeur_par_lot

##Nom des constantes
SCREAMING_SNAKE_CASE

exemple : VALEUR_MAX

##Nom des méthodes
camelCase

exemple: calculerValeurTotale

##Nom des classes
PascalCase

exemple: TerrainResidentiel

##Indentation
4 espaces

##Déclaration des paramètres dans une classe

Les paramètres doivent être déclarés au début du fichier.

##Déclaration des constantes dans une classe

Les constantes doivent être déclarées à la fin du fichier.

##Déclaration des méthodes dans une classe

Les constructeurs doivent être déclarés en premier, ensuite les setters, ensuite les getters et pour finir les méthodes publics.

##Pour les autres normes, se fier à la documentation Java
[Normes Java](http://www.oracle.com/technetwork/java/codeconventions-150003.pdf)
